# XML Parser v5

Система обработки XML-файлов от поставщиков с преобразованием в единый JSON-формат.

## Требования

- PHP 7.0 или выше (совместим с PHP 8.x)
- Расширение `simplexml` (обычно включено по умолчанию)
- Веб-сервер (Apache, Nginx) или встроенный сервер PHP

## Быстрый старт

### 1. Запуск через встроенный сервер PHP

```bash
cd E:\Projects\parser_v5
php -S localhost:8080
```

Откройте в браузере: **http://localhost:8080**

### 2. Тестирование

1. Положите XML-файл поставщика "Мой агент" в папку `input/moyagent/`
2. Откройте веб-страницу и нажмите кнопку **"Запустить обработку"**
3. Результат (JSON) появится в папке `json/`
4. Обработанный XML переместится в `input/moyagent/Processed/`
5. В разделе логов вы увидите подробности обработки

### 3. Настройка cron (автоматический запуск)

**Linux:**
```
* * * * * /usr/bin/php /путь/к/проекту/process.php
```

**Windows (Планировщик задач):**
- Программа: `php.exe`
- Аргументы: `E:\Projects\parser_v5\process.php`
- Расписание: каждую минуту

Скрипт сам проверяет, прошёл ли настроенный интервал.

## Как добавить нового поставщика

Добавление нового поставщика требует **минимальных изменений** — достаточно создать один PHP-файл:

### Шаг 1. Создайте файл парсера

Создайте новый PHP-файл в папке `parsers/`, например `AmadeusParser.php`:

```php
<?php
require_once __DIR__ . '/../core/ParserInterface.php';

class AmadeusParser implements ParserInterface
{
    public function getSupplierFolder()
    {
        return 'amadeus';  // Имя папки в input/
    }

    public function getSupplierName()
    {
        return 'Amadeus';  // Название для логов
    }

    public function parse($xmlFilePath)
    {
        // Ваша логика парсинга XML -> ORDER
        // См. MoyAgentParser.php как образец
    }
}
```

### Шаг 2. Создайте папку для файлов

```
input/
  amadeus/
    Processed/
    Error/
```

**Готово!** Система автоматически обнаружит новый парсер при следующем запуске.

## Формат выходного JSON

Все парсеры формируют JSON в едином формате ORDER (спецификация RSTLS):

```json
{
  "UID": "uuid-заказа",
  "INVOICE_NUMBER": "номер заказа",
  "INVOICE_DATA": "20251013121600",
  "CLIENT": "код клиента",
  "PRODUCTS": [
    {
      "UID": "uuid-продукта",
      "PRODUCT_TYPE": {"NAME": "Авиабилет", "CODE": "000000001"},
      "NUMBER": "номер билета",
      "STATUS": "продажа",
      "TRAVELLER": "ФАМИЛИЯ ИМЯ",
      "TAXES": [...],
      "PAYMENTS": [...],
      "COMMISSIONS": [...],
      ...
    }
  ]
}
```

## Веб-интерфейс

- **Журнал событий** — логи обновляются автоматически каждые 3 секунды
- **Запуск обработки** — кнопка для немедленного запуска
- **Автообработка** — запуск по таймеру, пока страница открыта
- **Настройка интервала** — от 10 секунд до 24 часов
- **Очистка логов** — удаление всех записей
